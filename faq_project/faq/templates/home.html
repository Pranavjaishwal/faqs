Welcome to the FAQ Management System

This project is a Django-based FAQ management system that supports multi-language translations, a WYSIWYG editor, and a REST API for managing FAQs. It also includes caching for improved performance and a user-friendly admin interface.

Project Details

  • Admin Panel: http://127.0.0.1:8000/admin/
  • API Endpoint: http://127.0.0.1:8000/api/faqs/
  • Development Server: Running on port 8000
  • Redis Cache: Running on port 6379
